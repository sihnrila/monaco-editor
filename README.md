# Monaco Test - EPUB 편집기

React와 Monaco Editor를 기반으로 한 EPUB 편집기입니다. 실시간 미리보기, 구문 검증, 표준 검사 등의 기능을 제공합니다.

## 🚀 주요 기능

### 📝 편집 기능
- **Monaco Editor**: VS Code와 같은 강력한 코딩 경험
- **멀티 탭**: 여러 파일을 동시에 편집
- **실시간 미리보기**: 편집한 내용을 즉시 확인
- **파일 트리**: 프로젝트 구조를 한눈에 파악

### 🔧 포맷팅 도구
- **XHTML 태그 삽입**: 제목(h1-h6), 단락(p), 인용구(blockquote), 섹션(section) 등
- **리스트 생성**: 순서 있는 목록(ol), 순서 없는 목록(ul)
- **코드 블록**: 다양한 언어를 지원하는 코드 삽입
- **스마트 태그 감싸기**: 선택한 텍스트나 현재 태그를 지능적으로 감쌈

### ✅ 검증 기능
- **실시간 구문 검증**: 타이핑하는 동안 자동으로 오류 검사
- **표준 점검**: EPUB 표준 준수 여부 확인
- **접근성 점검**: 웹 접근성 가이드라인 검사
- **오류 하이라이팅**: 코드에서 오류 위치를 시각적으로 표시

### 📊 로그 뷰어
- **탭별 분류**: 구문 오류, 전자책 표준 오류, 접근성 표준 오류
- **실시간 업데이트**: 구문 오류는 실시간으로 갱신
- **토글 가능**: 최소화/확장 상태 전환

## 🛠 설치 및 실행

### 필요 조건
- Node.js 14.0.0 이상
- npm 또는 yarn

### 설치
```bash
# 프로젝트 클론
git clone <repository-url>
cd monaco-test

# 의존성 설치
npm install
```

### 개발 서버 실행
```bash
npm start
```
브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속

### 프로덕션 빌드
```bash
npm run build
```

## 📖 사용법

### 1. 프로젝트 생성/열기
- **새 프로젝트**: 홈페이지에서 "새 프로젝트 만들기" 클릭
- **기존 프로젝트**: 프로젝트 목록에서 편집할 프로젝트 선택

### 2. 파일 편집
- **파일 열기**: 왼쪽 파일 트리에서 파일 클릭
- **멀티 탭**: 여러 파일을 동시에 열어 편집 가능
- **자동 저장**: 편집 내용은 자동으로 임시 저장됨

### 3. 포맷팅 도구 사용
상단 툴바에서 다양한 포맷팅 도구를 사용할 수 있습니다:

#### 텍스트 선택 후 버튼 클릭
```html
<!-- 텍스트 선택 후 "제목 1" 버튼 클릭 -->
안녕하세요 → <h1>안녕하세요</h1>
```

#### 커서 위치에서 버튼 클릭
```html
<!-- 커서가 태그 내부에 있을 때 버튼 클릭 -->
<p>안녕하세요</p> → <h1><p>안녕하세요</p></h1>

<!-- 빈 줄에서 버튼 클릭 -->
(빈 줄) → <h1></h1>
```

### 4. 검증 기능
#### 임시저장 (실시간 구문 검증)
- 편집 중 자동으로 구문 오류 검사
- 하단 로그 뷰어의 "구문 오류" 탭에서 확인
- 에디터에서 오류 위치 하이라이팅

#### 표준점검
- 상단 "표준점검" 버튼 클릭
- EPUB 표준 준수 여부 확인
- "전자책 표준 오류" 탭에서 결과 확인

#### 접근성 점검
- 상단 "접근성 점검" 버튼 클릭
- 웹 접근성 가이드라인 검사
- "접근성 표준 오류" 탭에서 결과 확인

### 5. 서버 저장
- 상단 "서버저장" 버튼 클릭
- 변경된 파일들을 서버에 저장
- 구문 오류만 검사 (표준 검사는 별도)

### 6. 미리보기
- 오른쪽 미리보기 패널에서 실시간 확인
- CSS 스타일이 적용된 상태로 렌더링
- 외부 CSS 링크도 자동 적용

## 🎨 사용자 인터페이스

### 헤더
- **프로젝트 이름**: 현재 편집 중인 프로젝트
- **버튼들**: 임시저장, 서버저장, 표준점검, 접근성 점검

### 메인 영역
- **왼쪽**: 파일 트리 (컨텍스트 메뉴로 파일/폴더 관리)
- **중앙**: Monaco Editor (멀티 탭 지원)
- **오른쪽**: 실시간 미리보기

### 하단 로그 뷰어
- **구문 오류**: 실시간 구문 검증 결과
- **전자책 표준 오류**: 표준점검 결과
- **접근성 표준 오류**: 접근성 점검 결과
- **토글 버튼**: 최소화/확장 상태 전환

## ⚡ 단축키

### 일반
- `Ctrl/Cmd + S`: 임시저장
- `Ctrl/Cmd + Shift + S`: 서버저장

### Monaco Editor 기본 단축키
- `Ctrl/Cmd + F`: 찾기
- `Ctrl/Cmd + H`: 바꾸기
- `Ctrl/Cmd + /`: 주석 토글
- `Ctrl/Cmd + D`: 다음 같은 단어 선택
- `Alt + 위/아래`: 줄 이동

## 🐛 문제 해결

### 로그가 표시되지 않음
1. 브라우저 콘솔에서 오류 메시지 확인
2. 로그 뷰어 토글 버튼 클릭하여 재열기
3. 페이지 새로고침 후 다시 시도

### 미리보기가 작동하지 않음
1. 파일이 올바른 XHTML 형식인지 확인
2. CSS 링크가 올바른지 확인
3. 브라우저 콘솔에서 네트워크 오류 확인

### 태그 감싸기가 이상함
1. 커서가 올바른 위치에 있는지 확인
2. 태그가 올바르게 닫혀있는지 확인
3. 브라우저 콘솔에서 디버그 로그 확인
